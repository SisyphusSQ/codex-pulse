## 失败处理

| 失败点 | 停止条件 | 记录方式 | 恢复 / 重跑方式 |
| --- | --- | --- | --- |
| framework checksum/version 不符 | 立即停止 | 记录 expected/actual digest 或 version，不记录完整下载 URL | 检查官方 release truth；不要复用坏 cache |
| tagged compile/load 失败 | 立即停止 | 记录 clang/dyld 摘要 | 检查 framework cache、build tag 和测试 rpath |
| package dependency/rpath/codesign 失败 | 立即停止 | 保留 ignored bundle 与 verifier 摘要 | 修复 build/assemble 后整条 package 重跑 |
| live smoke 退出或残留 | 停止并升级 | 精确 PID 与 content-free log 摘要 | 先停止 exact child，再诊断；不得批量 kill |
