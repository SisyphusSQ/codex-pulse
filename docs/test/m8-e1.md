| 失败点 | 停止条件 | 记录方式 | 恢复 / 重跑 |

| sink isolation | metrics 错误改变业务 writer/query 或丢失待确认 latency | 记录固定故障点和 dropped count | 修复 accumulator restore/maintenance admission 后 repeat/race |
