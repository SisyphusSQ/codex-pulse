## Implementation review rework

独立 subagent 首轮给出两个 blocking P2 与一个 advisory P3：

1. hidden inset titlebar 没有 Wails `--wails-draggable: drag`，窗口移动入口不可靠；
2. `ShellView` 外层 `aria-live=polite` 与内部 `role=status/alert` 嵌套，VoiceOver 可能重复播报；
3. app icon 的 `alt=Codex Pulse` 与相邻可见应用名重复播报。

修复采用 RED/GREEN：新增测试后先读回 4 个预期失败，再实现 `.wails-drag-region`、交互后代/`.wails-no-drag`、移除外层 live region，并把品牌图标改为装饰性空 `alt`。修复后 App/design focused 2 files/7 tests、typecheck、production build 与 diff check通过；重新打包后，在 ready态标题区先清除选择再执行 Computer Use drag gesture，应用保持 `/overview`、`darwin`、`zh-CN` ready state且没有文本选择/导航副作用。原 reviewer逐项复审三项均 closed，独立复验 focused/typecheck/diff通过，最终 `remaining_findings=0`、`blocking_findings=0`、`IMPLEMENTATION_REVIEW_PASS:YES`。
