## Keyboard / Assistive Semantics Checks

1. Tab 首次进入 skip link；激活后焦点落在主内容。
2. sidebar links 支持 ArrowUp/ArrowDown/Home/End；焦点环在 default/contrast/forced-colors 可见。
3. route 切换后滚动归零且页面标题获得编程焦点，读屏不重复播报整个 sidebar。
4. Local Status 危险确认对话框 Tab/Shift+Tab 留在 dialog，Escape/取消回到触发按钮。
5. 模拟 reduced-motion/reduced-transparency/contrast/forced-colors，动画和玻璃效果不会阻断内容辨认。
6. VoiceOver 快速检查六页面 landmark、heading、label、status 与 error retry accessible name。
