## Acceptance Matrix

| 面 | 验证入口 | 通过条件 |
| --- | --- | --- |
| evaluator | `internal/health` table tests | finite components/rules；priority/unknown/not_configured；持续窗口/缺口/边界确定 |
| migration | store migration tests | fresh/v13→v14、固定 checksum、旧行搬运、canonical readback、failure rollback |
| event plan | evaluator + store tests | finite exact reason；deterministic ID/fingerprint；fact replay no-op；later count；clear resolve；later reopen |
| atomic write | store failure injection | apply 全事务；冲突/取消/缺引用/写失败 rollback；非 evaluator event 不被 resolve |
| snapshot | store tests | metrics/lifecycle/last progress/last failure 在一个 read transaction；并发 writer 不混合 |
| service | health race tests | robfig immediate/30s、shared overlap、panic isolation、stale last projection、in-flight Close |
| app | app composition tests | metrics→health start；health→metrics close；业务 query/index/quota failure 隔离 |
| privacy | static/schema scan | 无 raw error/body/path/token/open labels；不新增 health 表/列，raw SQL 只限 v14 DDL |

## 失败与恢复

| 失败点 | 停止条件 | 恢复 |
| --- | --- | --- |
| rule | 同输入不同结果、unknown 误报 healthy、单点越过 sustained window | 修复 table/gap/boundary 后 focused repeat |
| event apply | 部分 observe/resolve、非 managed event 被关闭、replay 增 count | rollback fixture 后重跑 reopen/atomic matrix |
| snapshot | lifecycle/metrics 跨提交混合或 timestamp 缺失 | 修复单 read transaction 与 typed timestamp |
| service | overlap、panic 后不再调度、last projection 丢失、Close 泄漏 | fake cron/clock/sink race 重跑 |
| app | health failure 改变主业务启动/关闭或 Store 先关 | composition failure injection + race |
