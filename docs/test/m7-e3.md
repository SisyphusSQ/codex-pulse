## 目标

验证 `/sessions` 只消费 generated Wails query DTO，支持时间、项目、模型、activity、排序与稳定 keyset 分页，能够从 URL 恢复有限筛选、当前列表 cursor 和选中 Session，并在独立详情区展示安全 metadata、聚合用量、pricing evidence 与 bounded content-free Turn timeline。

成功标准：

- Vue Router query 是可恢复 UI 状态源；未知 key、数组、空值和非法枚举 fail closed，并替换为规范 URL。
- list cursor 只原样进入 URL；筛选或排序变化清空 cursor 和 selection。Turn AEAD cursor 只保留在当前 UI 生命周期，不解析、不写 URL、不持久化。
- 服务端顺序、totals、activity、pricing、partial 与 timeline 顺序不在 Vue 重算；微美元只做 locale 展示换算。
- unknown 显示 `--`，真实 0 保留；loading、known empty、partial、stale、fatal、not-found 与 cursor recovery 各有明确入口。
- 页面不显示 Session/Turn opaque ID、prompt/response/reasoning 正文、tool output、路径、raw model、offset、generation 或底层 error cause。
- source 与 implementation 在同一 1440×1024 viewport 对照，900×600 和 nominal 1120×720 不溢出且主路径可用。
